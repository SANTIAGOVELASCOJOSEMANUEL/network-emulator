# 🌐 Simulador de Red Web
https://santiagovelascojosemanuel.github.io/network-emulator/
<img width="1919" height="975" alt="image" src="https://github.com/user-attachments/assets/4a1f8a5f-6152-4910-a0b0-5ec38a4b297b" />

Un **simulador de topologías de red basado en navegador** desarrollado con **HTML, CSS y JavaScript**, que permite crear, conectar y probar dispositivos de red dentro de un entorno visual interactivo.

Este proyecto busca ofrecer una experiencia similar a herramientas como **Packet Tracer**, pero ejecutándose directamente en el navegador.

---

## 🚀 Características

* 🖥️ Creación de dispositivos de red (PC, Router, ISP, AP, etc.)
* 🔌 Conexión de dispositivos mediante cables
* 🌐 Configuración de direcciones IP
* 📡 Simulación de conectividad entre dispositivos
* 🧪 Pruebas de **ping entre PCs**
* 🖱️ Interfaz gráfica interactiva con **Canvas**
* 🔍 Soporte para **zoom y navegación en la topología**
* 📟 Consola de simulación integrada

---

## 🛠️ Tecnologías utilizadas

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **HTML5 Canvas API**

## ⚙️ Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/SANTIAGOVELASCOJOSEMANUEL/emulador-de-red.git
```

2. Entrar al directorio:

```bash
cd emulador-de-red
```

3. Abrir el archivo:

```
index.html
```

en tu navegador.

No requiere instalación ni dependencias externas.

1. Agregar dispositivos a la topología

## 🎯 Objetivo del proyecto

Este simulador fue creado con fines:

* Educativos
* Experimentales
* Aprendizaje sobre redes y simulación gráfica

El objetivo es construir una herramienta web que permita **visualizar y entender cómo funcionan las redes de computadoras**.
